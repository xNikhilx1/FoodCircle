# 🍴 FoodCircle

FoodCircle is a simple frontend project built with **HTML and CSS**.  
It provides pages for browsing, viewing, and creating recipes, along with a recipe maker page.  

## 📄 Pages
- Home Page  
- Recipe Details Page  
- Create/Edit Recipe Page  
- Recipe Maker Page

- CSS3
